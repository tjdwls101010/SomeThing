# Google AI Studio 프론트엔드 디자인 프롬프트 가이드

Google AI Studio에서 정교한 프론트엔드 디자인을 생성하기 위한 프롬프트 작성 가이드.

---

## 핵심 원칙

### 1. 타이포그래피 조합 명시하기

**프롬프트 예시:**
```
세리프와 산세리프 폰트 조합을 사용해줘:
- 제목: 우아함을 위해 [Playfair Display / Syncopate / Cormorant] 사용
- 본문: 가독성을 위해 [Inter / Space Grotesk / DM Sans] 사용
```

**중요한 이유:**
- 단일 폰트는 단조로움을 유발한다.
- 세리프는 권위와 우아함을, 산세리프는 현대적인 가독성을 전달한다.
- 다양한 폰트 두께(300, 400, 500, 600, 700)를 요청하는 것이 좋다.

---

### 2. 풍부한 색상 팔레트 정의하기

**프롬프트 예시:**
```
풍부한 색상 팔레트로 디자인해줘:
- 주요 색상(Primary): CTA 및 강조를 위한 [특정 헥스 코드]
- 보조 색상(Secondary): 보조 요소를 위한 [특정 헥스 코드]
- 배경 색상 변형: 최소 3가지 음영
- 의미별 색상(Semantic): 성공, 경고, 정보 등 뚜렷한 색조로 구분
```

**피해야 할 표현:**
- "모던한 색상 조합을 사용해줘" (너무 모호함)
- 주요 색상 하나만 지정하는 것

**권장 사항:**
- 최소 5개 이상의 색상을 지정한다.
- 각 색상의 용도(배경, 텍스트, 강조, 오버레이 등)를 정의한다.

---

### 3. 애니메이션 라이브러리 지정하기

**프롬프트 예시:**
```
Framer Motion을 사용한 애니메이션:
- 자연스러운 움직임을 위한 스프링(Spring) 물리 효과
- 스크롤 트리거 애니메이션
- 자식 요소들에 대한 순차적(Staggered) 애니메이션
- 여러 속성(크기, 투명도, 필터)을 포함한 호버(Hover) 효과
```

**지속 시간 가이드라인:**
```
애니메이션 지속 시간:
- 마이크로 인터랙션: 300-500ms
- 페이지 전환: 800-1200ms
- 배경 애니메이션: 10-30s (무한 반복)
```

**피해야 할 사항:**
- 기본적인 Tailwind 애니메이션(animate-pulse, animate-bounce)만 사용하는 것
- 0.3초 미만의 짧은 애니메이션

---

### 4. 배경 효과 상세히 설명하기

**프롬프트 예시:**
```
다음과 같은 동적인 배경을 만들어줘:
- 움직이는 그라데이션 블롭 (blur-[40-60px], mix-blend-screen)
- 그레인/노이즈 텍스처 오버레이
- 깊이감을 위한 비네트(Vignette) 효과
- 다양한 투명도를 가진 여러 레이어
```

**고급 요청:**
```
배경에 포함할 요소:
- 독립적인 애니메이션 경로를 가진 3개의 떠다니는 블롭 요소
- 별이 빛나는 필드 또는 파티클 시스템
- 중앙에서 퍼져나가는 방사형 그라데이션 오버레이
```

---

### 5. 이미지 필터 처리 요청하기

**프롬프트 예시:**
```
시각적 통일감을 위해 이미지 필터를 적용해줘:
- 기본 상태: grayscale-[0.2], sepia-[0.1]
- 호버 상태: grayscale-0, scale-105
- 오버레이: 브랜드 색상과 mix-blend-multiply 적용
```

**효과:**
- 이미지가 디자인에 자연스럽게 녹아든다.
- 호버 시 이미지에 "생기"가 돌아오는 듯한 느낌을 준다.

---

### 6. 다중 속성 호버 효과 요청하기

**프롬프트 예시:**
```
호버 효과는 여러 속성을 동시에 애니메이션 처리해야 해:
- 크기 변환 (1.02-1.1)
- 투명도 변경
- 색상/필터 전환
- 텍스트 위치 이동
- 배경색 변경
- 그림자 강화
```

**피해야 할 사항:**
- 단일 속성만 변경하는 것 (예: hover:scale-105만 사용)
- 모든 요소에 동일한 호버 효과를 사용하는 것

---

### 7. 성능 최적화 명시하기

**프롬프트 예시:**
```
성능을 최적화해줘:
- 애니메이션 요소에 will-change 사용
- transform: translateZ(0)를 이용한 GPU 가속
- 배경의 블러 강도 줄이기 (최대 40-60px)
- 패시브(passive) 이벤트 리스너 구현
```

---

### 8. 인터랙티브 요소 요청하기

**프롬프트 예시:**
```
고급 인터랙션을 포함해줘:
- 호버 가능한 요소에 반응하는 커스텀 커서
- 키보드 네비게이션 지원 (화살표 키, Esc)
- 스크롤 진행률 표시기
- 스크롤에 따른 패럴랙스(Parallax) 효과
```

## 프롬프트 템플릿

### 기본 템플릿

```
다음 명세에 따라 [프로젝트 타입]을 만들어줘:

**타이포그래피:**
- 제목 폰트: [폰트 이름] (세리프/디스플레이)
- 본문 폰트: [폰트 이름] (산세리프)
- 사용할 두께: 300, 400, 500, 600, 700

**색상 팔레트:**
- 주요 색상: #[hex] - [용도]
- 보조 색상: #[hex] - [용도]
- 배경: #[hex] (2-3가지 변형 포함)
- 강조: #[hex]
- 텍스트: #[hex] (주요), #[hex] (보조)

**애니메이션 (Framer Motion 사용):**
- 페이지 전환: 800ms ease-out
- 호버 효과: 300-500ms, 스프링 물리 효과 적용
- 배경 애니메이션: 15-25s 무한 반복
- 스크롤 시 페이드인-업(fade-in-up) 애니메이션

**시각 효과:**
- 배경에 움직이는 그라데이션 블롭
- 이미지 필터: 약간의 그레이스케일/세피아, 호버 시 제거
- 오버레이에 백드롭 블러(backdrop blur)
- 깊이감을 위한 mix-blend-mode

**컴포넌트 포함 사항:**
- 다중 속성 호버 상태
- 목록에 순차적 애니메이션 적용
- 부드러운 스크롤 네비게이션
- 모바일 반응형 디자인

**성능:**
- 애니메이션에 GPU 가속
- 최적화된 블러 값
- 이미지 지연 로딩(Lazy loading)
```

### 고급 템플릿 (최고 수준의 정교함)

```
정교한 디자인의 프리미엄 [프로젝트 타입]을 만들어줘:

**타이포그래피 시스템:**
- 디스플레이: 히어로 텍스트용 [Syncopate/Playfair Display]
- 제목: [Space Grotesk/Cormorant]
- 본문: [Inter/DM Sans]
- 유동적 타이포그래피(clamp) 구현

**풍부한 색상 시스템:**
- 특정 용도를 가진 8개 이상의 색상 정의
- 그라데이션 조합 포함
- OKLCH 색상을 사용한 다크/라이트 모드 지원

**애니메이션 아키텍처:**
- Framer Motion의 variants 시스템 활용
- 물리 기반 애니메이션을 위한 useSpring
- 패럴랙스를 위한 useScroll + useTransform
- 마운트/언마운트 시를 위한 AnimatePresence
- 커스텀 이징(easing) 곡선

**고급 시각 효과:**
- 3개 레이어의 움직이는 블롭 배경
- 노이즈/그레인 텍스처 오버레이 (SVG)
- 비네트 그라데이션
- 스프링 트래킹을 이용한 커스텀 마우스 커서
- 그레이스케일에서 컬러로 전환되는 이미지 효과

**마이크로 인터랙션:**
- 호버 시 버튼 빛/쓸기 효과
- 그림자가 점점 짙어지며 카드가 들리는 효과
- 텍스트 스크램블/타이프라이터 효과
- 로딩 상태 애니메이션 (통통 튀는 점들)

**접근성 및 성능:**
- 키보드 네비게이션 (화살표, Esc)
- focus-visible 상태
- will-change 최적화
- 패시브 이벤트 리스너
- 지연 애니메이션을 위한 Intersection Observer
```

## 참고: v0 vs Google AI Studio 비교

| 요소 | v0 기본 결과물 | Google AI Studio 최적화 결과물 |
|---|---|---|
| 폰트 | 단일 Geist 폰트 | 세리프 + 산세리프 조합 |
| 색상 | 1개의 주요 색상 반복 | 5개 이상의 다양한 팔레트 |
| 애니메이션 | 기본 Tailwind | 고급 Framer Motion |
| 호버 | translate-y만 적용 | scale + opacity + filter + shadow |
| 배경 | 단순한 블러 블롭 | 다중 레이어 블롭 + 그레인 + 비네트 |
| 지속 시간 | 0.3s | 0.5-2s |

이 가이드를 사용하면 v0에 비해 Google AI Studio에서 훨씬 더 정교한 결과물을 생성하는 데 도움이 될 것입니다.
