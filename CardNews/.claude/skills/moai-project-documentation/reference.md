# Reference

## Documentation File Types

### Essential Files (Required)

| File | Purpose | Audience | Maintenance |
|------|---------|----------|-------------|
| README.md | Project overview, quick start | Users + Developers | Updated with features |
| CHANGELOG.md | Version history | Users + Developers | Every release |
| LICENSE | Legal terms | Legal + Users | Rarely changes |
| CONTRIBUTING.md | Development guidelines | Developers | As needed |
