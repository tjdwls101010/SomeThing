## 2025 Updates & Breaking Changes

**Prometheus 3.x Breaking Changes**:
- Agent mode now stable (CLI flag changes)
- Native histograms enabled by default in certain contexts
- Remote Write 2.0 requires server-side support

**Grafana 11.3 Breaking Changes**:
- Scenes-based dashboards may have different JSON schema
- Some legacy panel options deprecated
- PDF export format improved

**OpenTelemetry 1.33 Status**:
- Logs, metrics, traces all GA
- Python, JavaScript, Java SDKs stable
- Semantic conventions finalized

**Loki 3.x Considerations**:
- Bloom filters still experimental (use with caution in critical paths)
- Schema migration required from older versions
- Tenant ID isolation recommended for multi-tenancy

**Elasticsearch 8.17 Deprecations**:
- ILM now default for new indices
- Legacy template syntax deprecated
- Upgrade required from 7.x
